# Local-Train-Booking-WebApp
Skills : Html, CSS , JavaScript , PHP 
 
Developed a Local Train Booking website that allows users to search, book, and 
manage train reservations using HTML, CSS, JavaScript, PHP, and MySQL. 
Implemented features for real-time train schedule viewing, and booking 
confirmation, providing an efficient and user-friendly booking experience. 
Utilized PHP and MySQL for backend processing and database management, 
ensuring secure storage and retrieval of user data and booking details. Deployed 
the website on a free cloud platform, ensuring accessibility and scalability.
